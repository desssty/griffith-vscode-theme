## Installation

1. Clone this repository:

```bash
git clone https://github.com/desssty/griffith-theme.git
cd griffith-theme
```

2. Package the `.vsix` file

   - Install `vsce` if you don‚Äôt have it yet:

     `npm install -g @vscode/vsce`

   - Then package the extension:

     `vsce package`

3. Restart VS Code.

4. Open the Command Palette (`Ctrl+Shift+P`), then choose:

   - `Preferences: Color Theme ‚Üí Griffith-Castle`
